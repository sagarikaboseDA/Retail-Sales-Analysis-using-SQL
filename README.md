# Brazilian E-Commerce Sales Analysis using-SQL
Dataset link : https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

# Objective
The main objective behind performing e-commerce analysis on the given dataset is to highlight valuable insights essential for making informed business decisions. The dataset comprises of multiple tables spanning over various inputs like Customer details, Geolocation, Order items, Order payment methods, Order reviews, Orders olaced, Product details, Seller details and Product category. The entire dataset comprises of over 100K+ order details between the year 2016 to 2018.
